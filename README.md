## Introduction
In this project, we will be analyzing trends in movies and TV shows over the years. We will utilize various datasets to extract insights regarding viewership, ratings, genres, and more.

## Objectives
- To analyze the performance of different movies and TV shows.
- To visualize trends in genres over time.
- To understand audience preferences and viewership statistics.

## Installation
To run this project locally, please follow the instructions below:
1. Clone this repository:
   ```
   git clone https://github.com/rohithay/film-analysis.git
   ```
2. Navigate into the project directory:
   ```
   cd film-analysis
   ```
3. Install required libraries:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the Jupyter Notebooks, execute the following command in the terminal:
```bash
jupyter notebook
```

This will open up the Jupyter interface in your browser. You can then navigate to the `notebooks/` folder to find the analysis and visualization notebooks.
